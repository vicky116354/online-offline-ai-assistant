# Online and Offline AI Assistant

## Description
This project is an AI Assistant that works both online and offline.
Offline mode works without internet using local logic.
Online mode can be extended with real AI models.

## Features
- Online and Offline modes
- Simple GUI using Tkinter
- Local database using SQLite
- Python-based MVP project

## Technologies Used
- Python
- Tkinter
- SQLite
